What is HTML

HTML is the standard markup language for creating Web pages. Hypertext Markup Language is a Language that supports HTML markup It provides structure to the content appearing on a wedsite,such as images,text,or videos by creating a basic skeleton.

HTML Basics

HTML (HyperText Markup Language) is the code that is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.

HTML Versions

HTML5 is the latest version of HTML.HTML has evolved with the passage of time.Other versions of HTML includes HTML 1.0,HTML 2.0,HTML 3.2,HTML 4.01 and XHTML.
